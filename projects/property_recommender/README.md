# Infrastructure

### Elasticsearch

This requires a local elasticsearch cluster.  Please see `athenaeum/infrastructure/elasticsearch` for details on how to set one up.

### Data formats

At the moment we only ingest csv files.

- Cost requirements
- Bedrooms etc
- Square footage requirements
- Room requirements
- Pet requirements
